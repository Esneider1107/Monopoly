# **Monopoly**

***Linea de comandos*** 
 - Jugar     -> Tirar dados y avanzar

 -  ComprarHotel -> Comprar un hotel en una propiedad
 -  Hipotecar   -> Hipotecar una propiedad
 -  Deshipotecar-> Deshipotecar una propiedad
